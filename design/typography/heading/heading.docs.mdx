The `Heading` component enhances content structure and readability with customizable levels and styles.

## Basic Usage

This example demonstrates a basic H1 heading:

```jsx
<Heading level={1}>Pioneering Mars: Our Grand Vision</Heading>
```
## Visual Level Customization
Here, an H2 heading is visually styled as an H1, maintaining semantic correctness while adapting the visual appearance:
```jsx
<Heading level={2} visualLevel={1}>Core Technologies for Martian Habitation</Heading>
```
## All Heading Levels
A hierarchical demonstration of all heading levels, showcasing the component's versatility in structuring content:
```jsx
<Heading level={1}>Project Ares: Roadmap to Mars</Heading>
<Heading level={2}>Phase I: Terraforming Infrastructure</Heading>
<Heading level={3}>Module 1.A: Atmospheric Processors</Heading>

<Heading level={5}>Component Spec: Advanced Zeolite Filters</Heading>
<Heading level={6}>Manufacturer Note: Requires enhanced radiation shielding</Heading>
