The `Image` component enhances Corporate Inc.'s website by providing a flexible and visually appealing way to display images. It supports responsive sizing, loading strategies, and visual enhancements like rounded corners and shadows.

## Contained Launch System Image
Here, an artistic depiction of a rocket launch trail symbolizes Corporate Inc.'s launch capabilities. The example uses `objectFit="contain"` and custom styles for background color, alongside `borderRadius` and `shadow` adjustments.
```jsx
<Image
  src="https://images.unsplash.com/photo-1516801967339-cb50a3ce7781?ixid=M3w3MDc2NDF8MHwxfHNlYXJjaHw5fHxzcGFjZSUyMHRlY2hub2xvZ3klMjBjb3Jwb3JhdGV8ZW58MXwwfHxibHVlfDE3NDcyMjMxNDV8MA&ixlib=rb-4.1.0"

  width="450"
  height="300"
  objectFit="contain"
  borderRadius="small"
  shadow="none"
  style={{ backgroundColor: 'var(--colors-surface-tertiary)' }}
/>
