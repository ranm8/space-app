A base component for constructing consistent icon components, ensuring a unified style across the Corporate Inc. website. It supports sizing, styling, and click events, tailored for space technology themes.

## Usage

To use the `Icon` component, you'll typically create specific icons using the `createIcon` function, providing the SVG content for each icon.

### Default Icons

Demonstrates the use of default icons with the standard size and inherited color.

```jsx
const StarSvgContent = () => (
  <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2l-2.81 6.63L2 9.24l5.46 4.73L5.82 21z" />
);
StarSvgContent.displayName = 'StarSymbol';
const StarIcon = createIcon(StarSvgContent);
<StarIcon />
```

### Interactive Icon
Showcases the implementation of an interactive icon with an `onClick` event handler, enhancing user engagement.
```jsx
const PlanetOrbitSvgContent = () => (
  <>
    <circle cx="12" cy="12" r="3" />
    <path d="M12 2a10 10 0 0 0-9.16 14.16A9.94 9.94 0 0 0 12 22a10 10 0 0 0 9.16-7.84A9.94 9.94 0 0 0 12 2zm0 18c-4.41 0-8-3.59-8-8s3.59-8 8-8 8 3.59 8 8-3.59 8-8 8z" />
  </>
);
PlanetOrbitSvgContent.displayName = 'PlanetOrbitSymbol';
const PlanetOrbitIcon = createIcon(PlanetOrbitSvgContent);
function handleClick() {
  alert('Icon was clicked');
}
<PlanetOrbitIcon onClick={handleClick} />
