## Basic Spinner

A simple spinner to indicate loading.

```jsx
<Spinner />
```
## Spinner with Loading Text
Display a message to inform the user what is being loaded.
```jsx
<Spinner loadingText="Loading..." />
```
## Overlay Spinner
An spinner that covers its parent container.
```jsx
