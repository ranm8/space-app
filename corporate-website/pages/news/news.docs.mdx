## Usage

The `News` component displays featured news and a grid of recent articles, with category filtering and a "Load More" button for pagination.

## Examples

### Basic News Page

This example showcases the `News` component with default featured news and regular news items. It includes category filtering and a "Load More" button.

```jsx
<News />
```

### News Page Empty State
This example shows how the `News` component renders when there are no news articles available.
```jsx
<News
  featuredNews={[]}
  news={[]}
  categories={['All', 'NoNewsCategory']}
  pageTitle="No Current News | Corporate Inc."
  pageDescription="Check back soon for the latest updates from Corporate Inc.'s Mars missions."
/>
