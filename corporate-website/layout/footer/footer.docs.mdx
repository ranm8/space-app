## Corporate Footer Component

A responsive footer component for corporate websites, featuring configurable link sections, copyright information, social media links, and attribution to Unsplash photos.

### Basic Footer

This example shows the basic usage of the `Footer` component with default settings.

```tsx
<Footer />
```
### Customized Footer
